# Renault Twizy Logotype
## Overview
This project provides a Renault Twizy Logotype created in Adobe Illustrator.  A SVG export is also included.

If the outline width is set to 2mm the resulting logotype should match exactly with the vinyl logotype sticker on the back of the Twizy.
